SMTP Per User

Send letters from Odoo using your own mail

Can configure different mail servers per user. Can specify email name for every server. For example - Email Name: 'My Custom Name' and sending mail is example@example.com. When sent, it will show like this: "My Custom Name example@example.com". Specific mail server can be set to ignore owner mail server and use forced server which was chosen by priority or by other means. This is forked and improved version of smpt-per-user for Odoo 11.0.
